## ファイル構成

Excelファイル内のシート構成:

- **MOS Excel 365 練習問題** - 知識問題の一覧表
- **回答一覧** - 知識問題の回答一覧表
- **Q01〜Q60** - 各知識問題の詳細シート
- **A01〜A60** - 各知識問題の回答・解説シート
- **実技問題一覧** - 実技問題の一覧表
- **P01〜P26** - 各実技問題の詳細シート
- **PA01〜PA26** - 各実技問題の回答・解説シート

## 使い方

1. `MOS-Excel-20260303.xlsx` をExcel (Microsoft 365) で開く
2. 「MOS Excel 365 練習問題」シートまたは「実技問題一覧」シートから問題を選択
3. 各問題シート (Q01, P01 等) で問題を確認
4. 自分で操作を試した後、回答シート (A01, PA01 等) で手順を確認

## 主なトピック

- データのインポート (CSV/テキスト)
- 検索・置換、ハイパーリンク
- ページ設定、印刷設定
- セルの書式設定、条件付き書式
- テーブル操作、フィルター、並べ替え
- 関数 (SUM, IF, VLOOKUP, XLOOKUP, COUNTIF, SUMIF, 動的配列関数 等)
- グラフの作成と編集
- ピボットテーブル
